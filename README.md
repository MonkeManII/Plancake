# Plancake
A lightweight binary serialization framework for C#.
> Plancake does not define any serializers on its own except for the ones in the
> [test](https://github.com/MonkeManII/Plancake/tree/main/MSTest) and [example](https://github.com/MonkeManII/Plancake/tree/main/example) directories.
> Instead, it is intended to be used as a library to define your own serializers.

## Getting Started
To get started, take a look at the [example](https://github.com/MonkeManII/Plancake/tree/main/example) folder for a basic example of how to use Plancake,
visit the [releases](https://github.com/MonkeManII/Plancake/releases) page to download the most recent library version,
or visit the [wiki](https://github.com/MonkeManII/Plancake/wiki) to learn about Plancake's basic functions.
